# GDB常用指令

### 1.打断点

```
break  xxx.c:第几行
break <位置> if <条件表达式> ： 位置可以是xxx.c:第几行， 函数名
例如：  
	break loop_function if i == 100
	break xxx.c:25 if i == 0

n  跳到下一行

s  进入该函数执行

c  往下继续执行

ctrl C  中断程序
```



### 2.打印变量值

```
print 变量名
```



### 3.改变变量值

```
set variable  变量名 = xx
```



### 4.bt指令

用于显示程序当前的调用栈，即程序执行到断点或错误时的函数调用链。这是调试崩溃、递归错误或异常行为的核心工具。



### 5.jump指令

用于强制程序跳转到指定位置继续执行

```
jump <文件名>:<行号>

jump <文件名>:<函数名>
```



### 6.display

跟踪查看一个变量，每次停下来都显示它的值



```
display    变量名

undisplay  变量名   // 取消追踪
```
